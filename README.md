# min_nettside

Statisk nettside bygget for Cloudflare Pages med flere undersider og delt design.

## Struktur

- `index.html` - Hjem
- `om-oss.html` - Om oss
- `prosjekter.html` - Prosjekter / tjenester
- `kontakt.html` - Kontakt oss
- `portal.html` - Logg inn til portal
- `registrer.html` - Opprett konto
- `verktoy.html` - Tjenester i portalen
- `dev-tools.html` - DEV tools
- `official-tools.html` - Official tools
- `prosjekter-portal.html` - Prosjekter i portal
- `live-support.html` - Live support
- `titan-ai.html` - Titan AI
- `beta-titan-ai.html` - Beta Titan AI
- `assets/styles/main.css` - Felles styling
- `assets/scripts/app.js` - Felles logikk og UI
- `assets/images/` - Logoer og ikoner
- `assets/fonts/` - Goodman-fonten

## Filer du kan bytte ut

Legg disse filene i `assets/images/` hvis du vil erstatte plassholderne med eksportene dine:

- `brand-mark.svg`
- `titan-mark.svg`
- `linkedin.svg`
- `instagram.svg`
- `github.svg`

Legg Goodman-fonten i `assets/fonts/` som:

- `Goodman-Regular.woff2`

Hvis fontfilen mangler brukes fallback-font automatisk.
